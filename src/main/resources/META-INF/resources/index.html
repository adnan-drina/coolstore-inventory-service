Coolstore Inventory Service

Product inventory for the Coolstore retail domain — stock levels per store location and availability lookups by item.

EndpointDescription
GET /api/inventory List all inventory items
GET /api/inventory/{itemId} One item by its itemId
GET /api/inventory/{itemId}/availability Availability summary for an item
GET /q/health Health report (liveness and readiness)
GET /q/metrics Prometheus metrics
